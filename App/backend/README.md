# How to run backend

1st, install the dependencies: <br/>
-> cd ./App/backend <br/>
-> npm install <br/>

2nd, setup .env file <br/>
-> cd ./App/backend <br/>
-> create .env file <br/>
-> copy what's inside default.env and pase into .env file <br/>
-> replace the boilerplate variables with what's in discord <br/>

3rd, run it <br/>
-> cd ./App/backend <br/>
-> npm start <br/>
