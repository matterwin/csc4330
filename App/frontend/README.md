# How to run frontend

1st, install the dependencies: <br/>
-> cd ./App/frontend <br/>
-> npm install <br/>

2nd, setup api variable <br/>
-> ./src/api/apiConfigDefault <br/>
-> follow those steps in that file <br/>

3rd, run it <br/>
-> cd ./App/frontend <br/>
-> npx expo start <br/>
